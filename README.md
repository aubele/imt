Dieses Projekt beinhaltet die Ergebnisse meiner Masterarbeit mit dem Thema:
Automatic Measurement of Intima-Media Thickness using Convolutional Neural Networks.

Das Repository ist aufgeteilt in die drei verfolgte Ansätze (one pixel, three pixel and region approach), 
welche die verwendete Grundwahrheit jeweils abändern.

Jeder Ansatz enthält drei Notebooks für Pre processing, U-Net Model und Post processing, sowie die finalen 
trainierten Models.

Der verwendete Datensatz ist nicht mit inbegriffen.
